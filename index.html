Creating virtual host in Apache

  1. /etc/hosts file in Notepad or Any Editor like Sublime Text or Notepad++ for Mac users open Finder then navigate to Go > Go To Folder now type /etc/hosts open hosts file. For Windows users C:\Windows\System32\Drivers\etc\hosts
  2. This opens a system file that contains the following line: -- 127.0.0.1 localhost
  3. On the next line, type 127.0.0.0 followed by a space and the name of the virtual host you want to create for example mysite.dev as new line will be127.0.0.1 mysite.dev
  4. Save the file when you have finished adding the names of your virtual hosts.
  5. Now open the main Apache configuration file, httpd.conf, which is located at Applications/MAMP/conf/apache/httpd.conf in MAC or C:\wamp\bin\apache\Apache2.2.11\conf\httpd.conf in Windows.
  6. Now search following line -- # Include /Applications/MAMP/conf/apache/extra/httpd-vhosts.conf
  7. Remove the hash (#) sign from the beginning of the line that begins with Include.
  8. Save the file, and then open Applications/MAMP/conf/apache/extra/httpd-vhosts.conf. This is where you define the virtual hosts
  9. At the bottom of the page are two examples of how to define virtual hosts in Apache. Now Update the files with below code.
<VirtualHost *:80>
    DocumentRoot "/Applications/MAMP/htdocs"
    ServerName localhost
</VirtualHost>
<VirtualHost *:80>
    DocumentRoot "/Applications/MAMP/htdocs/mysite"
    ServerName mysite.dev
</VirtualHost>

Save all the files you have edited, and restart the servers in the MAMP control panel. You should now be able to access the virtual host with the following URL: http://mysite.dev/

Note:- If you want to create more than one virtual host, copy one of the definitions, and edit it accordingly.
